# Contact API

APIs available in this service

```
// Retrieve all contacts
GET /contacts

// Get all contacts
POST /contacts

// Retrieve a single contact by id
GET /contacts/:id

// Update a single contact by id
PUT /contacts/:id

// Remove a single contact by id
DELETE /contacts/:id
```

Tools used

1. `expressjs`
2. `moogoosejs`
